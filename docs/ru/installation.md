# Полная установка

Подробное руководство по установке Paperclip + NemoClaw на Ubuntu VPS.

## Системные требования

| Параметр | Минимум | Рекомендуется |
|----------|---------|---------------|
| ОС | Ubuntu 22.04 | Ubuntu 24.04 |
| RAM | 8 ГБ | 16 ГБ |
| Диск | 20 ГБ свободного места | 50 ГБ |
| Node.js | 20+ | 22 (устанавливается автоматически) |
| pnpm | 9 | 9 (устанавливается автоматически) |

## Получение NVIDIA API-ключа

1. Перейдите на [build.nvidia.com](https://build.nvidia.com)
2. Зарегистрируйтесь (или войдите в существующий аккаунт)
3. Откройте **API Keys** в меню пользователя
4. Нажмите **Generate API Key**
5. Скопируйте ключ -- он понадобится при запуске `nemoclaw onboard`

## Запуск установки

```bash
sudo bash scripts/client-install.sh
```

Скрипт требует права root (`sudo`).

## Что делает скрипт

Установка проходит в 8 этапов:

1. **Проверка предусловий** -- ОС, RAM (>= 8 ГБ), диск (>= 20 ГБ в `/opt`), интернет
2. **Node.js** -- установка через nvm (версия 22) с проверкой целостности установщика
3. **pnpm** -- установка пакетного менеджера (pnpm@9)
4. **Paperclip** -- клонирование в `/opt/paperclip`, установка зависимостей, сборка
5. **NemoClaw** -- клонирование в `/opt/nemoclaw`, запуск `install.sh`
6. **Системный пользователь** -- создание пользователя `paperclip` для запуска сервиса
7. **systemd-сервис** -- создание `paperclip.service` (порт 3100)
8. **Запуск** -- активация и добавление в автозагрузку

## Проверка установки

### Health endpoint

```bash
curl -sf http://localhost:3100/api/health
```

Ответ `200 OK` означает, что Paperclip работает.

### Статус сервиса

```bash
systemctl status paperclip
```

Должен быть `active (running)`.

### Настройка NemoClaw

После установки выполните onboard:

```bash
nemoclaw onboard
```

Введите NVIDIA API-ключ при запросе.

## Решение проблем при установке

### Node.js не найден

Если скрипт не нашёл Node.js или версия ниже 20:

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
source ~/.bashrc
nvm install 22
```

### pnpm build завершается ошибкой

Убедитесь, что версия Node.js >= 20:

```bash
node --version
```

Если версия устарела, обновите через nvm:

```bash
nvm install 22
nvm use 22
```

### systemctl не запускает сервис

Проверьте логи:

```bash
journalctl -u paperclip -n 50 --no-pager
```

Частые причины:
- Не собран проект (`pnpm build` в `/opt/paperclip`)
- Не хватает прав у пользователя `paperclip` на `/opt/paperclip`
- Порт 3100 занят другим процессом

## Удалённый доступ

Порт 3100 не следует открывать напрямую в интернет без аутентификации.

Для безопасного удалённого доступа рекомендуется [Tailscale](https://tailscale.com) -- приватная VPN-сеть без настройки firewall. После установки Tailscale на VPS и ваш компьютер, Paperclip будет доступен по адресу:

```
http://<tailscale-ip>:3100
```

# Быстрый старт

Запуск платформы Paperclip + NemoClaw за 5 минут.

## Требования

| Параметр | Минимум |
|----------|---------|
| ОС | Ubuntu 22.04+ |
| RAM | 8 ГБ |
| Диск | 20 ГБ свободного места |
| Сеть | Доступ к nvidia.com |

## Установка

```bash
sudo bash scripts/client-install.sh
```

Скрипт установит Node.js, pnpm, Paperclip и NemoClaw автоматически.

## Настройка NemoClaw

```bash
nemoclaw onboard
```

При запуске потребуется NVIDIA API-ключ. Получите его на [build.nvidia.com](https://build.nvidia.com):

1. Зарегистрируйтесь или войдите
2. Перейдите в раздел API Keys
3. Создайте ключ и скопируйте его

## Запуск

Откройте в браузере:

```
http://localhost:3100
```

## Первые шаги

1. Нажмите **Новая компания** и введите название
2. Добавьте агентов (или импортируйте готовый шаблон)
3. Создайте задачу и назначьте её агенту
4. Наблюдайте за выполнением в разделе запусков
